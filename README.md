
# Dashboard de vulnérabilité climatique à Montréal

Ce projet présente un tableau de bord web interactif développé dans le cadre d’un essai de maîtrise en géomatique appliquée et télédétection.

Le dashboard vise à représenter la vulnérabilité aux vagues de chaleur à Montréal à partir de données spatiales, sociodémographiques, de végétation et de projections de mortalité.

## Technologies utilisées

- HTML
- CSS
- JavaScript
- OpenLayers
- ECharts
- Bootstrap
- Font Awesome
- GeoJSON

## Structure du projet

- `index.html` : structure principale de l’interface
- `css/style1.css` : mise en forme du tableau de bord
- `js/map1.js` : logique cartographique et interactions
- `data/` : données GeoJSON utilisées dans l’application
